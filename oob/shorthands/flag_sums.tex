\saNote{}
What we dub ``\textbf{common} precompiles'' are those precompiles that
can be given a mostly identical (common) treatment in the present module.
It does not refer to any other interpretation of ``common'' e.g. to
the frequency at which such precompiles are called upon \dots{}
Similarly what we dub ``\textbf{uncommon} precompiles'' are those precompiles
that evade this simple, uniform treatment.

\saNote{} 
If the \textsc{Ethereum} network configuration is at the \texttt{XXX} hard fork, one \emph{may} further impose
\[
	\locFlagSumYYYPrecompiles _{i} = 0 \qquad (\sanityCheck)
\]
thereby preventing \texttt{YYY}-specific precompiles to be executed, where \texttt{YYY} is every fork that is beyond \texttt{XXX}.
This isn't strictly speaking necessary, as the \trmMod{} won't recognize addresses in the set
$[\texttt{0x\,0b}..\texttt{0x\,11}] \cup [\texttt{0x\,100}]$
as precompile addresses, hence the sanity check mark.
