As per usual we declare a column \col{X} to be \textbf{counter-constant} if it satisfies
\[
	\If \ct _{i} \neq 0 ~ \Then \col{X}_{i} = \col{X}_{i - 1}.
\]
We impose that the following columns be counter-constant
\begin{multicols}{2}
	\begin{enumerate}
		\item $\relTxMax$
		\item $\coinbase\high$
		\item $\coinbase\low$
		\item \blockGasLimit
		\item \basefee
	\end{enumerate}
\end{multicols}
