The present \mmuMod{}-instruction allows one to move an arbitrary amount of data from one \textsc{ram} segment to another.
It is used by (message call) \inst{RETURN}'s, \inst{REVERT}'s, it is used for making the original copy of call data in a \inst{CALL} to the \inst{IDENTITY} precompile as well as for all (partial) copies of return data that feature at the end of \inst{CALL}'s to precompiles.
