
The constraints below impose that within the padding bytes\footnote{of which there are at least $\evmWordSize{}$ and at most $\evmWordSize + \llargeMO$} the \PBCB's are all zero. This is compatible with the final instruction of a code fragment being some \inst{PUSHX} instruction.
\begin{enumerate}
	\item \If $\CSR_{i} = 1$ \Then:
	\begin{enumerate}
		\item $\PBCB_{i}=0$
		\item $\IP_{i}=0$ \quad (\trash)
	\end{enumerate} 
\end{enumerate}
