
The constraints below impose that within the padding bytes\footnote{of which there are at least $\evmWordSize{}$ and at most $\evmWordSize + \llargeMO$} the \romColumnLimbByte's are all zero. This is compatible with the final instruction of a code fragment being some \inst{PUSHX} instruction.
\begin{enumerate}
	\item \If $\romColumnPadding _{i} = 1$ \Then:
	\begin{enumerate}
		\item $\romColumnLimbByte _{i} = 0$
		\item $\IP _{i} = 0$ \quad (\trash)
	\end{enumerate} 
\end{enumerate}
