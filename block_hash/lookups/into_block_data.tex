The purpose of the present lookup argument is to extract the true \inst{NUMBER} as defined by the \btcMod{} module.
The lookup is structured as follows:
\begin{description}
	\item[\underline{Selector:}] we use the $\isMacro$ column;
	\item[\underline{Source columns:}] ---
		\begin{multicols}{3}
			\begin{enumerate}
				\item \relBlock{}
				\item \absBlock{}
				\item \inst{NUMBER}
			\end{enumerate}
		\end{multicols}
	\item[\underline{Target columns:}] ---
		\begin{multicols}{3}
			\begin{enumerate}
				\item \relBlock{}
				\item \blockDataLo{}
				\item \INST{}
			\end{enumerate}
		\end{multicols}
\end{description}
\saNote{}
The numerical value of \inst{NUMBER} is \texttt{0x\,43}.
