and the account-row of the processing of an authority tuple.

		we impose
		\begin{enumerate}
		        \item \If $\accHasCode _{i + \roffAuthorizationProcessingAccountRow} = \false $ \Then

				\]

				\[
					\authAuthorityHasEmptyCodeOrIsDelegated _{i + (\roffAuthorizationProcessingAuthorityRow)} = \accIsDelegated _{i + \roffAuthorizationProcessingAccountRow}

		\end{enumerate}
